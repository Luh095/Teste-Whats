Porque escolher ver meu Portifólio?

Meu site de portfólio é a melhor forma de mostrar quem eu sou e o que faço de maneira profissional e organizada. Ele reúne meus principais trabalhos, projetos e conquistas, tudo apresentado de forma clara e atrativa. Além disso, é uma maneira de transmitir credibilidade e reforçar minha marca pessoal, tornando mais fácil compartilhar o que eu faço e me destacar no mercado.

Linguagens de Programação e Aplicativos

Front-End (HTML, CSS e JavaScript): é a interface visual e interativa que os usuários veem e utilizam. O trabalho de front-end envolve a criação e a otimização da interface para garantir uma experiência de usuário agradável e eficiente. O aplicativo Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft. Onde crie os códigos em HTML, CSS e JavaScript para o site ganhar vida e formar o que estão presenciando. E o aplicativo Figma é um editor gráfico de vetor e prototipagem de projetos de design baseado principalmente no navegador web, com ferramentas offline adicionais para aplicações desktop para GNU/Linux, macOS e Windows. Foi onde criei um esboço de como ficaria o site, seu design e a partir desse modelo criar este site.
